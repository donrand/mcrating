# 管理画面 CSV インポート 使い方ガイド

## 概要

「一括登録」ページの CSV インポート機能を使うと、大会の試合結果をまとめて登録できます。
1大会でも複数大会でも同じフォーマットで扱えます。

---

## CSVフォーマット

```
tournament_name,held_on,grade_coeff,mc_a,mc_b,winner[,round]
```

### 列の説明

| 列名            | 必須 | 説明                                             | 例                          |
|----------------|------|--------------------------------------------------|-----------------------------|
| tournament_name | 必須 | 大会名（同名の既存大会があれば自動で紐付け）       | UMB 2024 GRAND CHAMPIONSHIP |
| held_on         | 任意 | 開催日（YYYY-MM-DD 形式）。空欄可                | 2024-12-22                  |
| grade_coeff     | 必須 | 大会格係数（正の数値）                           | 3.0                         |
| mc_a            | 必須 | MC A の名前                                      | R-指定                      |
| mc_b            | 必須 | MC B の名前                                      | 呂布カルマ                  |
| winner          | 必須 | 勝者（下表参照）                                 | a                           |
| round           | 任意 | ラウンド名（下表参照）。空欄可                   | 決勝                        |

---

## winner の入力値

| 入力値               | 意味         |
|----------------------|--------------|
| a / a側 / 1          | MC A の勝利  |
| b / b側 / 2          | MC B の勝利  |
| draw / 引き分け / 0  | 引き分け     |

大文字・小文字・前後のスペースは無視されます。

---

## ラウンド名（推奨）

レーティングは開催日 → ラウンド順に計算されます。
以下の名称を使うと正しい順序で並び替えられます。

1. 1回戦
2. シード戦
3. 2回戦
4. ベスト16
5. ベスト8
6. 準決勝
7. 決勝

上記以外の名称でも登録できますが、ラウンド順ソートの対象外になります（末尾扱い）。

---

## grade_coeff（大会格係数）の目安

| 係数 | 対象                        |
|------|-----------------------------|
| 1.0  | 草大会・予選                |
| 1.5  | 地方大会                    |
| 2.0  | 中規模大会                  |
| 2.5  | 全国規模                    |
| 3.0  | UMB・KOK・戦極など主要大会  |

0.1 単位で任意の値を設定できます。
格係数管理ページから後から変更することも可能です（変更後は全再計算が必要）。

---

## サンプルCSV

### 単一大会（1大会のみ）

```csv
tournament_name,held_on,grade_coeff,mc_a,mc_b,winner,round
UMB 2024 GRAND CHAMPIONSHIP,2024-12-22,3.0,R-指定,呂布カルマ,a,決勝
UMB 2024 GRAND CHAMPIONSHIP,2024-12-22,3.0,MOL53,CIMA,b,準決勝
UMB 2024 GRAND CHAMPIONSHIP,2024-12-22,3.0,T-PABLOW,晋平太,a,準決勝
UMB 2024 GRAND CHAMPIONSHIP,2024-12-22,3.0,R-指定,MOL53,a,ベスト8
UMB 2024 GRAND CHAMPIONSHIP,2024-12-22,3.0,呂布カルマ,CIMA,a,ベスト8
```

### 複数大会（同じファイルに混在）

```csv
tournament_name,held_on,grade_coeff,mc_a,mc_b,winner,round
UMB 2024 GRAND CHAMPIONSHIP,2024-12-22,3.0,R-指定,呂布カルマ,a,決勝
UMB 2024 GRAND CHAMPIONSHIP,2024-12-22,3.0,MOL53,CIMA,b,準決勝
KOK 2024 FINAL,2024-11-03,3.0,T-PABLOW,晋平太,a,決勝
KOK 2024 FINAL,2024-11-03,3.0,舐達麻,FORK,b,準決勝
地方大会 2024,2024-10-15,1.5,MC太郎,MC次郎,a,決勝
```

大会名が同じ行は自動でグループ化されます。
既存の大会名と一致する場合は既存大会に紐付けられます（大文字・小文字を区別しません）。

## インポート手順

1. **CSVを用意する**
   上記フォーマットに従ってCSVを作成します。
   ExcelやGoogleスプレッドシートで作成してCSV形式で保存するか、テキストエディタで直接作成してください。

2. **一括登録ページを開く**
   管理画面の「一括登録」タブを開き、「CSV インポート」セクションの「開く」をクリックします。

3. **CSVを貼り付けまたはファイル選択**
   テキストエリアに直接貼り付けるか、「ファイルを選択」からCSVファイルをアップロードします。

4. **「読み込む」をクリック**
   パースエラーがある場合は赤字で表示されます。
   エラーがなければ大会プレビューが表示されます。

5. **プレビューを確認して登録**
   大会名・開催日・格係数・バトル件数を確認し、「○件を一括登録する」ボタンで登録します。

---

## 注意事項

- **MC名の一致**: MC名は既存のMCと完全一致（大文字小文字不問）する場合に自動で紐付けられます。
  一致しない場合は新規MCとして登録されます。

- **別名義の統合**: 同一人物の別名義がある場合は、登録後に「全再計算」が必要です。
  名義統合は `scripts/merge_mcs.mjs` の `MERGE_GROUPS` に追加して実行してください。

- **全再計算について**: 登録済みの大会にバトルを追加した場合や、格係数を変更した場合は、
  バトル管理ページの「全再計算を実行」ボタンを押してレーティングを再計算してください。

- **文字コード**: CSVの文字コードはUTF-8で保存してください。
  Shift-JISでは文字化けする場合があります。

- **コメント行**: `#` で始まる行はコメントとして無視されます。
  メモや区切りとして活用できます。

---

## 手動入力（CSVなし）

「大会設定（単一大会）」セクションでは、CSVを使わず直接フォームに入力して登録することもできます。

- 既存の大会を選択するか、新規大会名を入力
- 開催日・大会格係数を設定
- バトルを1行ずつ入力して「一括登録する」
